# Phase 2: Non-Linear Models & Testing

Building on the cleaned dataset from Phase 1, this phase focuses on testing more complex, non-linear machine learning algorithms to see if they can capture the salary patterns better.

We trained two new models and compared their R2 and RMSE scores. We also built an interactive script to test how the model handles live, manual inputs.

**Models tested:**
* K-Nearest Neighbors (KNN)
* Decision Tree

**Files in this folder:**
* `Phase2_KNN_Model.ipynb` - Training and saving the KNN model.
* `Phase2_DecisionTree_Model.ipynb` - Training and saving the Decision Tree model.
* `Phase2_Interactive_Input.ipynb` - A script to manually enter employee details and test the KNN prediction.
* `cleaned_employee_dataset.csv` - The dataset carried over from Phase 1.
* `Phase_2_Report.docx` - Our detailed report for this phase.
