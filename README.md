## Project Overview
In many organizations, deciding a fair and consistent salary for employees is a critical but difficult task. If this decision relies solely on guesswork or manual analysis, it can lead to bias and inconsistency. 

This Project-Based Learning (PBL) assignment aims to solve this by developing a regression-based machine learning system. Using a synthetically generated dataset of 500 employees with 20 distinct features (such as experience, performance rating, leadership score, and job role), this system predicts a fair, data-driven salary.
